void printLCD ( )

display normalized acceleration data on LCD

printLCD() writes the normalized accelerometer values from the gy-521 module to a 16x2 LCD. Normalized values correspond to the parameter space of our SuperCollider processing (scrub and reverb wet/dry) and much more useful to see than raw data.

◆ readAcc()

void readAcc ( )

read raw x, y, z accelerometer values from gy-521 and print to serial monitor

readAcc() reads the raw data from the gy-521 module. x, y, and z correspond to roll, pitch, and net acceleration, respectively. The function writes these values to the serial monitor sequentially, each followed by a character, 'x', 'y', or 'z' to designate its origin to whatever software may be parsing the serial data.
