void rt2 ( )

play "Close Encounters" melody three times

rt2() plays the "Close Encounters" melody three times on an external buzzer. The driving signal is generated via CTC PWM from timer/counter 4. Sleeps for REST_TIME seconds (1.5 in this example). BEAT_DUR <\tt> sets the duration of a single note in milliseconds.

FFT setup.

rt3p0() generates the random signal to be transformed via FFT. It initializes rt3p1() and rt4() , which read wall times and perform the FFT, respectively. rt3p0() then suspends itself.

◆ rt3p1()

void rt3p1 ( double *  randoms)

send signal to FFT and calculate wall time

rt3p1() sends the random signal, *randoms, to the FFT, then reads the time in milliseconds the FFT takes to compute. The function repeats this process five times, printing the current wall time to the serial monitor each iteration. The final value printed is the wall time. rt3p1() suspends itself.

Parameters
*randomsthe random signal to be sent to the FFT. Signal comprises N_RANDOMS random doubles in the range [0, 1].
Returns
returns void
